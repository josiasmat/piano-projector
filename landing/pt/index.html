Piano Projector

Um teclado de piano virtual versátil e fácil de usar para educação musical.

Abrir aplicativo online

O que é o Piano Projector?

Piano Projector é um aplicativo web completo que exibe um teclado de piano limpo e responsivo e destaca as notas que você toca em um teclado MIDI conectado. Foi projetado para educadores musicais, estudantes e músicos — perfeito para projeção em sala de aula, aulas online e demonstrações musicais.

Como Usar

  • Abra o aplicativo usando o botão acima.
  • Clique no botão "Controle" para selecionar um dispositivo de entrada.
  • Se solicitado, conceda permissão para acesso MIDI.
  • Toque em seu controlador MIDI ou no teclado do computador para ver as notas sendo destacadas.
  • Arraste o teclado do piano usando o mouse para mudar sua posição vertical e role a tela para aumentar ou reduzir o zoom.
  • Você pode redimensionar a janela do navegador ou usar tela cheia para projeção em sala de aula.

Recursos

  • Suporte a entrada MIDI.
  • Reconhece entrada de pedais ("sustain", sostenuto e una corda).
  • Tamanho e cores do teclado ajustáveis.
  • Diversos tipos de rótulos e marcadores coloridos para as teclas.
  • Suporte a transposição.
  • Sons de piano, órgão e cravo (opcional).
  • Funciona offline (baseado em tecnologia PWA).
  • Leve e de código aberto.

Atalhos de teclado

  • F2 - alterna modo de rotulagem.
  • F3 - alterna modo de adesivos.
  • F4 - define a tônica para o formato de rótulo dó-móvel.
  • Ctrl+clique - define rótulo/adesivo para todas as oitavas.
  • F9 - alterna visibilidade da barra de ferramentas.
  • F11 - entrar em tela cheia.
  • ALT - abre menu de navegação por teclado.
  • ESC - aciona o "pânico" MIDI.

Compatível com:

  • Mozilla Firefox
  • Google Chrome
  • Microsoft Edge
  • Qualquer navegador baseado em Chromium

Não é compatível com Apple Safari.

Funciona melhor em computadores desktop ou laptop.
