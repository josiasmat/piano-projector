Piano Projector

Un teclado de piano virtual versátil y fácil de usar para educación musical.

Abrir aplicación en línea

¿Qué es Piano Projector?

Piano Projector es una aplicación web completa que muestra un teclado de piano limpio y responsivo y destaca las notas que tocas en un teclado MIDI conectado. Fue diseñada para educadores musicales, estudiantes y músicos — perfecta para proyección en aula, clases en línea y demostraciones musicales.

Cómo Usar

  • Abre la aplicación usando el botón de arriba.
  • Haz clic en el botón "Control" para seleccionar un dispositivo de entrada.
  • Si se solicita, otorga permiso para acceso MIDI.
  • Toca tu controlador MIDI o el teclado de la computadora para ver las notas siendo destacadas.
  • Arrastra el teclado del piano usando el ratón para cambiar su posición vertical y desplázate en la pantalla para ampliar o reducir el zoom.
  • Puedes redimensionar la ventana del navegador o usar pantalla completa para proyección en aula.

Características

  • Soporte de entrada MIDI.
  • Reconoce entrada de pedales ("sustain", sostenuto y una corda).
  • Tamaño y colores del teclado ajustables.
  • Varios tipos de etiquetas de teclas y marcadores de color.
  • Soporte de transposición.
  • Sonidos de piano, órgano y clavecín (opcional).
  • Funciona sin conexión (basado en tecnología PWA).
  • Ligero y de código abierto.

Atajos de teclado

  • F2 - alterna modo de etiquetado.
  • F3 - alterna modo de marcación.
  • F4 - define la tónica para el formato de etiqueta do-móvil.
  • Ctrl+clic - define etiqueta/marcador para todas las octavas.
  • F9 - alterna visibilidad de la barra de herramientas.
  • F11 - entra en pantalla completa.
  • ALT - abre menú de navegación por teclado.
  • ESC - activa el "pánico" MIDI.

Compatible con:

  • Mozilla Firefox
  • Google Chrome
  • Microsoft Edge
  • Cualquier navegador basado en Chromium

No es compatible con Apple Safari.

Funciona mejor en computadoras de escritorio o portátiles.
